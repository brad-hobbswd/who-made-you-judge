# Who Made You Judge?
**Rediscovering Truth Without Condemnation**
An interactive online edition of **Who Made You Judge?**, written by **Bradley Hobbs**. This website presents the complete manuscript in a clean, book inspired reading experience while exploring what Scripture teaches about judgment, discernment, grace, accountability, and restoration through Jesus Christ.
## Live Website
https://brad-hobbswd.github.io/who-made-you-judge/
## About
*Who Made You Judge?* invites readers to move beyond common misunderstandings surrounding judgment and return to the example of Jesus Christ. Rather than promoting condemnation, the book encourages believers to pursue biblical truth with humility, mercy, and compassion.
The website is designed to provide a distraction free reading experience that feels like opening the pages of a printed book while remaining fully responsive across desktop, tablet, and mobile devices.

## Design Goals
This project was created to provide readers with an immersive online reading experience inspired by traditional printed books while maintaining modern web accessibility and responsive design principles.
The layout emphasizes readability through generous spacing, elegant typography, and intuitive navigation so readers can focus on the message rather than the interface.
